# Centri — Compliance Assessment

> Know where you stand. Fix what's broken.

Guided compliance self-assessment tool for federal contractors and GovCon vendors. No backend, no login, no consultant required.

## Frameworks

| Framework | Controls | Questions |
|-----------|----------|-----------|
| CMMC Level 1 | 17 | 51 |
| NIST CSF 2.0 | 18 | 54 |
| NIST 800-53 Rev 5 | 30 | 90 |
| FedRAMP Tailored (LI-SaaS) | 24 | 72 |

## Features

- Plain-English questions mapped to real framework controls
- Yes / No / Partial / N/A answers with real-time scoring
- Prioritized gap report sorted by risk (Critical → High → Medium)
- Actionable remediation steps for every failing control
- PWA-ready — installable as a desktop or mobile app
- Zero backend — runs entirely in the browser, works offline after first load
